# MPP Data Analysis Portfolio

This repository showcases selected coursework and applied data analysis projects from my Master of Public Policy program at the University of Chicago Harris School of Public Policy.  
Each folder corresponds to a core quantitative course emphasizing statistical modeling, program evaluation, or machine learning.

The projects demonstrate my ability to:
- Clean, merge, and analyze data in **R** and **Python**
- Conduct hypothesis testing, regression analysis, and causal inference
- Apply simulation, matching, IV, DiD, and event study methods
- Communicate results through reproducible markdown reports and visualizations

---

##  Course Folders

### [Advanced Statistics for Data Analysis I](./Advanced-Statistics-I)
- Hypothesis testing and simulation-based inference  
- Matching methods and causal interpretation  

### [Advanced Statistics for Data Analysis II](./Advanced-Statistics-II)
- Binary outcome models (LPM, probit/logit)  
- Instrumental variables and causal identification  

### [Advanced Program Evaluation](./Advanced-Program-Evaluation)
- Difference-in-Differences (DiD) and event study designs  
- Multiple hypothesis testing and statistical significance adjustment  

### [Machine Learning](./Machine-Learning)
- Supervised learning and classification  
- Discriminant analysis and model evaluation
